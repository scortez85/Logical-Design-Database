Add 3D plane to scene

Add Forklift to scene by selecting the model from the assets folder and dragging it into the scene

Set scale of Forklift to 0.1 on every vector and Zoom in to see it better

Set a new Texture to the plane by searching for every texture, then choose ground

Select Forklift and click add Component. Name the component

 ForkLiftMove, then choose a new C# script

Click create and add. Then double click component name to open in editor.
